# @hyperpipe/bridge

`@hyperpipe/bridge` is the first-party integration package shared by Hyperpipe Core,
Gateway, and client applications.

It contains the currently shared runtime surfaces for:

- auth helpers
- config helpers
- public gateway protocol and discovery helpers
- plugin manifest and packaging helpers
- small first-party shared UI helpers used by Hyperpipe clients

This package is intended for first-party Hyperpipe package consumption. It is not
positioned as a general-purpose standalone utility library.
